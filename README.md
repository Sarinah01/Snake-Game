# 🐍 Snake Game

A classic Snake Game made using **HTML**, **CSS**, and **JavaScript** as part of my **#100DaysOfCode Challenge (Day 15)**.

## 🎮 Game Features:
- Smooth keyboard controls (Arrow keys)
- Score tracking
- High Score saving in `localStorage`
- Responsive UI with CSS

## 🛠️ Technologies Used:
- HTML5
- CSS3
- JavaScript (ES6)
